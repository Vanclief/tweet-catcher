# tweet-catcher
A python utility for saving tweets in a time stamp database.


Example config:
```
database:


twitter_api:
  consumer_key: "XXX"
  consumer_secret: "XXX"
  access_token: "XXX-XXX"
  access_token_secret: "XXX"


languages:
  - en


topics:
  bitcoin:
    - 'btc'
    - '$btc'
    - '#btc'
    - 'bitcoin'
    - '$bitcoin'
    - '#bitcoin'
```
